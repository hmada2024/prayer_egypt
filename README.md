# Arabic Cities Database Creator (Egypt Focus)

## وصف المشروع

هذا المشروع هو سكريبت بايثون يقوم بإنشاء قاعدة بيانات SQLite تحتوي على معلومات تفصيلية حول المدن والقرى والتجمعات السكنية في دولة مصر. يتم استخلاص البيانات مباشرة من OpenStreetMap (OSM) عبر Overpass API، مما يلغي الحاجة إلى تحميل ملفات OSM ضخمة محليًا. تتضمن المعلومات المستخرجة أسماء المواقع باللغة العربية والإنجليزية (إذا كانت متاحة)، والإحداثيات الجغرافية، والمناطق الزمنية، واسم المحافظة باللغة العربية.

## المميزات الرئيسية

*   **استخلاص البيانات عبر الإنترنت:** يتم جلب البيانات مباشرة من Overpass API، مما يوفر الوقت والمساحة بتجنب تحميل ملفات OSM كبيرة.
*   **التركيز على دولة مصر:** تم تصميم السكريبت خصيصًا لاستخراج بيانات المدن والتقسيمات الإدارية في مصر فقط.
*   **توحيد الأسماء باللغة العربية:** يعطي السكريبت الأولوية لاستخدام الأسماء باللغة العربية (إذا كانت متوفرة في بيانات OSM).
*   **معلومات إدارية:** يتضمن اسم المحافظة (الولاية) باللغة العربية لكل موقع.
*   **تحديد المنطقة الزمنية:** يتم تحديد المنطقة الزمنية لكل موقع تلقائيًا باستخدام خطوط الطول والعرض.
*   **تخزين البيانات في SQLite:** يتم تخزين البيانات المستخرجة في قاعدة بيانات SQLite بسيطة وسهلة الاستخدام.
*   **شريط التقدم:** يعرض شريط تقدم مرئي أثناء معالجة البيانات.
*   **تسجيل الأخطاء:** يتم تسجيل معلومات حول سير العملية وأي أخطاء محتملة.

## المكتبات المستخدمة

*   **`sqlalchemy`:**
    *   **الدور:** إنشاء محرك اتصال بقاعدة بيانات SQLite وتنفيذ أوامر SQL لإنشاء الجدول وإضافة البيانات.
*   **`timezonefinder`:**
    *   **الدور:** تحديد المنطقة الزمنية التي يقع فيها موقع معين بناءً على خط الطول والعرض.
*   **`logging`:**
    *   **الدور:** تسجيل الأحداث والمعلومات والأخطاء التي تحدث أثناء تشغيل السكريبت.
*   **`pycountry`:**
    *   **الدور:** توفير معلومات حول الدول (يستخدم حاليًا فقط للتحقق من اسم الدولة ولكن يمكن استخدامه لأغراض أخرى مستقبلًا).
*   **`tqdm`:**
    *   **الدور:** عرض شريط تقدم مرئي أثناء معالجة البيانات.
*   **`requests`:**
    *   **الدور:** إرسال طلبات HTTP إلى Overpass API لجلب بيانات OpenStreetMap.
*   **`json`:**
    *   **الدور:** تحليل استجابة JSON المستلمة من Overpass API.
*   **`pandas`:**
    *   **الدور:** تحويل البيانات المستخرجة إلى DataFrame لتسهيل حفظها في قاعدة البيانات بكفاءة.

## كيفية التشغيل

1. **تأكد من تثبيت بايثون:** يجب أن يكون لديك بايثون مثبتًا على جهازك.
2. **إنشاء بيئة افتراضية (اختياري ولكنه موصى به):**
    ```bash
    python -m venv .venv
    # أو باستخدام virtualenv
    # virtualenv .venv
    ```
3. **تفعيل البيئة الافتراضية:**
    *   **على نظام Windows:** `.venv\Scripts\activate`
    *   **على نظام macOS و Linux:** `source .venv/bin/activate`
4. **تثبيت المكتبات المطلوبة:**
    ```bash
    pip install sqlalchemy timezonefinder pycountry tqdm requests pandas
    ```
5. **تشغيل السكريبت:**
    ```bash
    python create_arabic_cities_db.py
    ```
    (تأكد من أن اسم ملف السكريبت هو `create_arabic_cities_db.py` أو قم بتعديل الأمر وفقًا لذلك).

بعد تشغيل السكريبت بنجاح، سيتم إنشاء ملف قاعدة بيانات SQLite باسم `egyptian_cities_enhanced.db` في نفس المجلد الذي يوجد به السكريبت.
